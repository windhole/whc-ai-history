AIの研究・挑戦の歴史 1990 → 2026 / プログラマー視点の通史

36年を一文で言えば、人間がルールを書く時代から、データから関数を学習させる時代、 巨大な汎用モデルにプロンプトで指示する時代、そしてAIに目標を渡して自律的に働かせる時代への移行の歴史です。

1990s→人間がルールを書く → データから関数を学習させる
2000–10s→データから特徴量を学習させる（representation learning）
2020–23→巨大な汎用モデルにプロンプトで指示する
2024–26→目標を渡して自律的に働かせる（エージェント）
AI研究・挑戦のできごと 同時代の社会の主な出来事
第1期 / 01

記号処理から統計的学習へ 1990 – 2000

1980年代の「エキスパートシステム」ブームが崩壊し、第二次AIの冬に突入した時期。 記号処理（symbolic AI）から、データと統計に基づくアプローチへの地殻変動が起きた。 「ルールをハードコードする」発想から「データから関数を学習させる」発想へのパラダイムシフト。

1990
強学習可能性（Boosting）の理論的基礎。「弱い学習器の集合は強い学習器を作れる」ことが証明され、後のAdaBoost・勾配ブースティングの源流に。
keyRobert Schapire / Boosting world東西ドイツ統一、湾岸危機
1990
リカレントニューラルネット（RNN）の基礎研究。系列・時系列を扱う再帰構造が進展。記号処理派と接続主義（コネクショニズム）派の論争が続く。
keyJeffrey Elman / Elman network world日本のバブル経済が崩壊開始
1991
第二次AIの冬が深刻化。Lispマシン市場が崩壊し、エキスパートシステムの商業的失敗が明確に。資金が引き上げられる。
keyAI Winter / expert systems の凋落 worldソ連崩壊、WWW 一般公開
1992
TD-Gammon。バックギャモンをプレイする自己対戦型ニューラルネット。強化学習（TD学習）の実用的成功例として象徴的。
keyGerald Tesauro / 強化学習(TD) worldバルセロナ五輪、マーストリヒト条約
1993
強化学習が学問分野として確立へ。エージェントが報酬で学習する枠組みが整理される。
keySutton & Barto / agent, reward worldMosaic公開、商用ネットの本格化
1995
サポートベクターマシン（SVM）発表。カーネル法で非線形分類を可能にし、2000年代前半まで機械学習の主役に。理論的に美しく扱いやすかった。
keyVapnik, Cortes / SVM, カーネルトリック worldWindows 95、Amazon・eBay 創業
1995
ランダムフォレストの源流（Random Decision Forests）。決定木のアンサンブル学習が提案される。
keyTin Kam Ho / 決定木, アンサンブル worldJava言語 正式リリース
1997
Deep Blue が世界チェス王者カスパロフに勝利。AIが人間トップに公式戦で勝った象徴的事件。ただし機械学習ではなく総当たり探索＋評価関数の力業。
keyIBM Deep Blue / 探索, ブルートフォース world香港返還、京都議定書採択
1997
LSTM（長短期記憶）発表。RNNの「勾配消失問題」を解決するゲート機構。後の音声認識・機械翻訳・NLPの基盤に。この10年で最も後世に効いた発明の一つ。
keyHochreiter, Schmidhuber / LSTM, ゲート機構 world初代プリウス発売、Netflix 創業
1998
LeNet-5（CNN）論文発表。畳み込みニューラルネットで手書き数字認識（MNIST）を実用化。郵便番号読み取りなどに応用。2012年AlexNetの直接の祖先。
keyLeCun, Bengio / CNN, LeNet, MNIST worldGoogle 創業、Windows 98
1999
NVIDIA が GPU を発表（GeForce 256）。当時はゲーム用途だったが、後にニューラルネット学習を爆速化し深層学習革命の物理的前提となる。
keyNVIDIA / GPU, 並列計算 worldユーロ導入、Napster、Y2K対応
2000
確率的グラフィカルモデルの隆盛。ベイジアンネットワーク、隠れマルコフモデル（HMM）が音声認識などで主流化。データ駆動・確率論的アプローチが定着。
keyJudea Pearl / HMM, ベイジアンネット worldITバブル崩壊、PlayStation 2

// programmer's note

  • パラダイムの転換点。 if-then ルールを人間が書き下す記号処理から、損失関数を最小化して重みを最適化する統計的学習へ。「モデルを訓練する」発想がここで主流に。
  • 当時の主役は SVM。 理論保証があり、カーネルで非線形も扱え、小〜中規模データで安定。ニューラルネットは学習が不安定で重いと冷遇されていた。
  • 雌伏の10年。 LSTM(1997)・CNN/LeNet(1998) はほぼ完成していたのに、データ・GPU・学習テクニックが不足して本領を発揮できなかった。
  • Deep Blue の罠。 1997年の勝利は機械学習ではなく探索＋人手調整の評価関数。「賢くなった」のではなく「ハードで殴った」例。
第2期 / 02

ディープラーニング革命 2000 – 2020

出揃ったアルゴリズム（CNN・LSTM）に、ビッグデータ・GPU・学習テクニックという3つの燃料が揃い、ディープラーニングが爆発。 「人間が特徴量を設計する」時代から「データから特徴量そのものを学習する（representation learning）」時代へ。 後半は「学習済み巨大モデルを使う」時代の幕開け。

2006
ディープラーニング復活の号砲。Deep Belief Networks。層ごとの事前学習（pre-training）で深い多層NNの学習が初めて実用化。「deep learning」という語が広まる。
keyGeoffrey Hinton / DBN, 層ごと事前学習 worldTwitter 開始、Amazon AWS 開始
2009
ImageNet 公開。1400万枚規模のラベル付き画像DB。「アルゴリズムよりデータ」を象徴する基盤に。
keyFei-Fei Li / ImageNet, 大規模データセット world世界金融危機、Bitcoin 誕生
2010
ReLU 活性化関数が普及。max(0, x) という単純さで学習を約6倍高速化し、勾配を深い層まで流せるように。Kaggle 開始。
keyNair & Hinton / ReLU, Kaggle worldiPad 発売、欧州債務危機
2011
IBM Watson が Jeopardy! でクイズ王に勝利。自然言語処理＋情報検索＋機械学習の統合システム。
keyIBM Watson / NLP, 質問応答 world東日本大震災、アラブの春
2012
AlexNet が ImageNet コンペで圧勝。ディープCNN＋GPU＋大規模データが従来手法を蹴散らす。深層学習革命の起点。同年 Google Brain が「猫」を教師なしで認識。
keyKrizhevsky, Sutskever, Hinton / AlexNet, ILSVRC worldロンドン五輪、Facebook 上場
2013
word2vec。単語を意味ベクトルに変換し「king − man + woman ≒ queen」が成立。NLPの土台に。
keyTomas Mikolov / word2vec, embedding worldスノーデン事件（NSA監視暴露）
2014
GAN（敵対的生成ネットワーク）。生成器と識別器を競わせて本物そっくりのデータを生成。生成AIの源流。同年 VGGNet、DeepFace、seq2seq・Attention も進展。
keyIan Goodfellow / GAN, Attention機構 worldクリミア併合、エボラ流行
2015
ResNet（残差ネットワーク）。スキップ接続で152層という超深層を学習可能にし、画像認識で人間の精度（誤差5.1%）を超える（3.57%）。同年 OpenAI 設立、TensorFlow 公開。
keyKaiming He / ResNet, スキップ接続 worldパリ協定、ギリシャ財政危機
2016
AlphaGo が囲碁トップ棋士 李世乭に勝利。「あと10年は無理」と言われた囲碁を、深層強化学習＋モンテカルロ木探索で攻略。同年 WaveNet（音声合成）。
keyDeepMind, Hassabis / AlphaGo, 深層強化学習 worldBrexit国民投票、ポケモンGO
2017
Transformer 論文「Attention Is All You Need」。RNNを捨てて Attention だけで系列処理。並列化しやすく、現在のLLM（GPT・BERT）すべての基盤に。この20年で最重要の発明。
keyVaswani 他 / Transformer, Self-Attention world仮想通貨バブル、#MeToo運動
2018
BERT と GPT-1 登場。「巨大テキストで事前学習 → 個別タスクに微調整（fine-tuning）」という転移学習のパラダイムが確立。同年 AlphaFold 1。
keyGoogle(BERT), OpenAI(GPT) / 転移学習 world米中貿易摩擦、GDPR施行
2019
GPT-2。「危険すぎる」と段階的公開され話題に。大規模言語モデルのスケール則が見え始める。
keyOpenAI / GPT-2, スケーリング world香港デモ、新型コロナ確認（年末）
2020
GPT-3 発表（1750億パラメータ）。few-shot 学習で「プロンプトを書くだけ」で多様なタスクをこなし汎用性を実証。同年 AlphaFold 2 がタンパク質立体構造予測の50年来の難問をほぼ解決。
keyOpenAI(GPT-3), DeepMind(AlphaFold2) / LLM worldCOVID-19 パンデミック

// programmer's note

  • 2012年が分水嶺。 AlexNet以前は特徴量を手書きするのが当たり前。以降は「データを流してネットワークに特徴を見つけさせる」が標準に（Software 2.0）。
  • 3つの燃料が揃った。 アルゴリズム（90年代に完成）・データ（ImageNet 2009）・計算資源（GPU）が2012年前後で初めて揃った。発明が10年待って花開いた。
  • Transformer 一強へ。 画像はCNN・系列はRNN/LSTMという棲み分けが「とりあえずTransformer」に収束。スケーリング則が発見され巨大化競争が始まる。
  • 「作る」から「使う」へ。 GPT-3以降、AIは「自分で訓練するもの」から「APIを叩いてプロンプトで指示するもの」になった。
第3期 / 03

生成AI・対話AI・推論・エージェント 2020 – 2026

研究室の成果が一気に世間一般のツールへ転化。「モデルを訓練する」から「巨大な汎用モデルにプロンプトで指示する」へ、 さらに「AIに目標を渡してツールを使わせ自律的に作業させる（エージェント）」へと抽象度が一段ずつ上昇。 生成AI → 対話AI → 推論モデル → エージェント、という4段階の変化。

2020
GPT-3（1750億パラメータ）でfew-shot学習を実証。同年 AlphaFold 2 がタンパク質立体構造予測の難問をほぼ解決。
keyOpenAI(GPT-3), DeepMind / LLM, プロンプト worldCOVID-19、リモートワーク普及
2021
テキスト→画像生成の幕開け。CLIP（画像と言語を結びつける）とDALL·Eが登場。GitHub Copilot プレビュー公開で「AIがコードを書く」が現実に。
keyOpenAI / CLIP, DALL·E, GitHub Copilot world東京五輪（無観客）、半導体不足
2022
拡散モデルで画像生成が爆発（Stable Diffusion／Midjourney）。そして11月30日 ChatGPT 公開。2か月で1億ユーザーに達し史上最速の普及。AIが一般人の日常ツールになった転換点。
keyOpenAI(ChatGPT/GPT-3.5) / 拡散モデル, RLHF worldロシアのウクライナ侵攻、世界的インフレ
2023
GPT-4（マルチモーダル＝画像も入力可）。Meta が LLaMA をオープンソース公開し誰でも動かせるLLMが普及。Anthropic の Claude、Google の Gemini が対抗。AI開発の一時停止を求める公開書簡も。
keyGPT-4, LLaMA, Claude, Gemini / RAG, OSS LLM world生成AI著作権訴訟が多発
2024
GPT-4o（オムニ）で音声・画像・テキストをリアルタイム統合。そして9月 o1：推論モデル登場。答えを即出さず「思考の連鎖（chain-of-thought）」で順を追って考え自己修正する新クラス。
keyOpenAI(GPT-4o, o1) / 推論モデル, CoT worldNVIDIA時価総額が世界最高水準、EU AI Act
2025
推論モデルとエージェントの年。o3 が ARC-AGI で高スコア。中国勢の躍進（DeepSeek-R1 が低コストで推論性能、Qwen・Kimi・GLM 等）。実験的AIが数学オリンピックで金メダル級を達成。Webブラウズ・コード実行・ツール使用を自律的に行うエージェントが主流化。
keyo3, ChatGPT Agent, DeepSeek-R1 / エージェント world米中AI覇権競争、企業のAI導入が約8割
2026
（現在）エージェント基盤の本格的な企業導入フェーズ。コーディング・エンジニアリングを自律実行するエージェント（Codex系、Claude Code など）が実務に組み込まれる。推論とツール使用の統合がさらに進む。実験的な高性能モデルは安全上の懸念から限定提供の段階に。
keyエージェント基盤 / 自律コーディング, AIガバナンス world企業ワークフローの再設計、AI規制と運用統制

// programmer's note

  • 抽象度が4段階上がった。 ①生成AI（生成できる）→ ②対話AI（会話で指示できる）→ ③推論モデル（考えてから答える）→ ④エージェント（目標を渡すと完遂する）。
  • 「推論モデル」が最大の構造変化。 内部で長い思考過程を展開し、計算リソースを推論時（test-time）に多く使う。学習を大きくするだけでなく「考える時間を増やす」第二の軸が見つかった。
  • コーディングが主戦場に。 Copilot（補完, 2021）→ チャットでコード生成（2023）→ 自律コーディングエージェント（2025〜26）。ファイルを読み、テストを書き、実行し、修正までする。
  • オープンモデルと中国勢の台頭。 LLaMA公開以降クローズドAPI一強の構図が崩れ、ローカルで動く高性能モデルが選択肢に。DeepSeek-R1 が低コスト推論を実証。

全体総括：36年の一本の流れ

時代ごとに「開発者にとってAIとは何だったか」と「主要技術」を並べると、抽象度が一段ずつ上がっていく。

時期開発者にとってのAIとは主要技術
1990年代 人間がルールを書くもの → データから関数を学習させるもの SVM / 決定木 / （冷遇された）NN
2000–10年代 データから特徴量を学習させるもの CNN / RNN・LSTM / Transformer
2020–2023 巨大な汎用モデルにプロンプトで指示するもの LLM / 拡散モデル / RLHF
2024–2026 目標を渡して自律的に働かせるもの 推論モデル / エージェント
一貫して効いている法則：良いアルゴリズムは、データと計算資源が追いつくまで「待つ」。 LSTM・CNN が10年待って花開いたように、Transformer も「スケールさせる環境」が整って初めて真価を発揮した。
